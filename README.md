✨ Features
Gameplay

Smooth gravity-flip mechanic with instant feel
Progressive difficulty — speed and spawn rate ramp every ~6 seconds
Bonus collectible orbs with pulsing animation
Persistent high score via localStorage
Fully responsive — scales to any screen size

Visuals

Deep space background with 3-layer parallax star field
Procedural nebula blobs for atmosphere
Scrolling cyan grid overlay for depth
Glowing neon pillar lips at the gap edge
Player diamond with radial gradient, specular shine, and particle trail
Screen shake + colour flash on gravity flip, orb collect, and death
Warp ring animation on start and game-over screens
Staggered fade-in for the game-over UI

Audio (100% synthesised — zero audio files)

Gravity flip — rising sine chirp
Pipe pass — short transient click
Orb collect — 3-note ascending chime
Milestone (every 10 pipes) — 4-note arpeggio
Death — descending sawtooth buzz
Game start — 4-note ascending fanfare

All sounds generated in real time via the Web Audio API.
